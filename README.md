# Airbnb-CDMX-2025
Proyecto Final de Módulo 8: "Análisis y modelado de precios de Airbnb en CDMX". 

El proyecto se compone de un procesamiento y limpieza (data wrangling y detección de outliers) de la base de datos reales obtenidos de la página Inside Airbnb. La creación del modelo de regresión lineal múltiple para estimar precios en función de características del alojamiento y ubicación. Junto con la evaluación del modelo mediante análisis de residuos y comparación observado vs. predicho. Para finalizar con la construcción de un dashboard interactivo en Shiny con visualizaciones dinámicas, filtros y mapas geoespaciales (Leaflet), así cómo la generación de insights sobre factores clave de precio y concentración del mercado.
