# Airbnb-CDMX-2025
Proyecto Final de Módulo 8: "Análisis y modelado de precios de Airbnb en CDMX"
El siguiente proyecto se compone del
* Procesamiento y limpieza (data wrangling y detección de outliers) de la base de datos reales obtenidos de la página Inside Airbnb (https://insideairbnb.com/get-the-data/).
* Desarrollo de modelo de regresión lineal múltiple para estimar precios en función de características del alojamiento y ubicación.
* Evaluación del modelo mediante análisis de residuos y comparación observado vs. predicho.
* Construcción de dashboard interactivo en Shiny con visualizaciones dinámicas, filtros y mapas geoespaciales (Leaflet).
* Generación de insights sobre factores clave de precio y concentración del mercado.
